# Handoff — cc-storage-terminal

Дата: 2026-06-18

## Состояние
MVP + грид-редизайн (Create/стимпанк) готовы. 94 юнит-теста зелёные, smoke + render-preview зелёные.

## Грид-редизайн Create/стимпанк (2026-06-18)
Список-строки → **вид магазина**: сетка плиток-слотов (спрайт + счётчик + id), степпер количества.
Скин — Create/стимпанк (андезит-корпус + латунь-акцент), перекрывает белый DESIGN.md
(тот для сайтов; здесь монитор в MC + явная просьба на стимпанк).
- `src/sprites.lua` — НОВЫЙ: сикстант-символы CC (128-159), 1 клетка = 2×3 субпикселя.
  `M.cell` чистый энкодер (BR — опорный субпиксель для инверсии fg/bg), юнит-тест.
  7 спрайтов категорий 4×6 (Create=шестерёнка, Redstone=факел, …). `M.draw` = I/O.
- `src/render.lua` — переписан: палитра Create через `setPaletteColour` (андезит-серые +
  латунь/медь/парчмент + 7 цветов бейджей), грид плиток (beveled-корпус, спрайт цветом
  категории, `xN` счётчик, id снизу trunc), горизонт-чипы, степпер-кейпад оверлей
  (`- + +8 +64 Max Clear OK X`), индикатор `page X/Y`. **English ASCII** (CC-шрифт без кириллицы).
  `render.applyPalette(mon)` зовётся в startup.
- `src/ui_logic.lua` — `layout` (chips+grid на всю ширину), `gridDims`, `tiles`, `chips`, `stepper`.
- `src/startup.lua` — perPage-скролл, тач плиток/чипов, степпер-логика, `model.pressed`, палитра.
- `scripts/gen-installer.lua` — НОВЫЙ: переиспользуемый генератор бандла (читает `src/*.lua` +
  `*.cfg`, подбирает уровень длинных скобок). Запуск из корня: `lua5.1 scripts/gen-installer.lua`.
- Анимация тонкая: латунная рамка нажатой плитки, инверсия, скролл на страницу
  (попиксельный плавный скролл не делаем — лаг синка монитора в MP).
- Лимит движка: нет текстур MC на мониторе → спрайты на **категорию** (7), не на предмет.
- Глаз-чек раскладки: `lua5.1 test/render-preview.lua [W H]` (палитра/спрайты в превью не видны —
  мок не красит, sprites.draw подменён ASCII-плейсхолдером; цвета проверять in-game).

## Дальше — in-game проверка (за пользователем)
Расстановка: справа компа — Stock Ticker, сзади — склад, слева — Advanced Monitor.
Деплой: скопировать `src/*.lua` + `*.cfg` **плоско** в корень компа (`startup.lua` = автозапуск),
либо `pastebin`/`wget` по файлам, либо в `world/computercraft/computer/<id>/`. Ребут.

Чек-лист:
1. Скин: тёмный андезит-фон, латунные акценты, плитки-корпус (не мусор-глифы).
2. Грид ~6-8 колонок: спрайт категории + счётчик `xN` + id снизу, читаемо.
3. Чипы категорий в ряд (Create первым после All); тап фильтрует, активный = латунь.
4. Остатки обновляются ~2с.
5. Тап по полю поиска → фокус (латунь); печать фильтрует живьём; backspace стирает; enter снимает фокус.
6. Тап по полосе адреса переключает Main↔Core.
7. Скролл `[^]`/`[v]` листает страницами (`page X/Y`).
8. Тап по плитке → степпер-кейпад; `- + +8 +64 Max Clear` набирают число; OK → тост
   «Ordered Nx… -> <адрес>» + посылка; X закрывает.
9. Нажатие плитки → латунная рамка/инверсия (тонкий фидбэк), без лагов.

## Заметки реализации
- Все модули `return M`; единственный main-слой — `startup.lua`.
- Чистая логика (`ui_logic`, `stock`, `names`, `addresses`, `order`) юнит-тестируема без CC.
- Периферии авто-детект по типу (`Create_StockTicker`, `monitor`/`monitor_advanced`), override стороной в `config.lua`.
- Деплой кладёт файлы плоско → `require("config")` и т.п. резолвится из корня компа.
